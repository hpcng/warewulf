# Dependency Licenses

This project uses a number of dependencies, in accordance with their own
license terms. These dependencies are managed via the project `go.mod` and
`go.sum` files, and included in a `vendor/` directory in our official source
tarballs.

A full build or package of Warewulf uses all dependencies listed below.  If you
`import "github.com/hpcng/warewulf"` into your own project then you may use a
subset of them.
